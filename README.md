# النشاط الثاني: التنقل الأساسي (Basic Stack Navigation)

يوضح هذا المشروع كيفية التنقل بين الشاشات في Flutter باستخدام مفهوم الـ Stack.

## المتطلبات المنفذة:
- إنشاء شاشتين: HomeScreen و DetailScreen.
- استخدام Navigator.push() للانتقال للشاشة التالية.
- استخدام Navigator.pop() للعودة للشاشة السابقة.
- إضافة أزرار تحكم في كل شاشة.

## شرح عمل الـ Stack:
1. عند تشغيل التطبيق، يتم دفع HomeScreen إلى أسفل المكدس.
2. عند الضغط على زر الانتقال، يتم دفع DetailScreen فوق الشاشة الرئيسية.
3. عند الضغط على زر العودة، يتم إزالة DetailScreen من المكدس لتظهر الشاشة الرئيسية مرة أخرى.
